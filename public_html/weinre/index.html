what the heck is weinre?

Check out the latest docs: they explain everything:

http://people.apache.org/~pmuellr/weinre/docs/latest/

Install a recent version via npm:

sudo npm -g install weinre # or whatever

weinre is part of the Apache Cordova project.

Browse the list of weinre issues.

Open a bug or feature request against weinre.

Discuss weinre at the weinre group at Google Groups.
